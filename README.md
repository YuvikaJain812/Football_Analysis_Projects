# football-performance-analysis
Excel-based analysis projects using English Premier League data
### Darwin_nunez_analysis 
A project based upon analysis of frowards data of Liverpool FC across three seasons (2022/23, 2023/24 & 2024/25) to analyze effectivness and efficiency of Darwin Nunez in the team. 
Includes:
- Goals & Assists Comparison
- Per 90' metrics
- adjusted for Non-Penalty goals (NPGs)
- pivot tables based comparison with liverpool players
### LFC_General_analysis
Project based on analyzing squad performances of Liverpool across four seasons
- 2018/19
- 2019/20
- 2023/24
- 2024/25
And inter-seasonal comparisons to see what changed and how. Also includes:
- per 90' metrics
- position based contributions and comparisons
- inter-seasonal comparisons
- attacking contributions breakdown
### Mo_salah_analysis
A comprehensive project that collects and analyzes data of all big 6 attackers from 2017/18-2024/25 to evalute attacking performances 
Includes:
- Goals, assists, minutes, starts
- per 90' metrics
- team contribution and minutes share
- penalty-adjusted metrics
- pivot table comparisons
